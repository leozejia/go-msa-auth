# go-msa-auth

`go-msa-auth` 是一个使用 Go 编写的微服务，用于验证授权码的有效性。客户端通过 HTTP POST 请求与服务交互，服务会根据存储的授权码验证结果返回授权码是否有效。

---

## 功能

- 接收客户端发送的授权码。
- 验证授权码是否有效。
- 返回授权码的验证结果：
  - **有效**：返回 `{"status": "valid"}`
  - **无效**：返回 `{"status": "invalid"}`

---

## 技术栈

- **语言**：Go
- **Web 框架**：Gin
- **容器化**：Docker
- **依赖管理**：Go Modules
